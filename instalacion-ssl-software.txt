Python 3 

Módulo sklearn_cotraining-master.zip
Se instala con -> pip3 install sklearn_cotraining-master.zip, si no va con pip3 probar con pip.
Para probar la instalación cargar en Jupyter el notebook cotraining-python-test.ipynb, al ejecutarlo no debe salir ningún mensaje, señal de que el módulo ha sido importado correctamente.
